# Курсовой проект
## "Создание компьютерной игры на движке Unity"
### Название игры: "Delusion"



#### Дерево файлов:
```bash
├── Assets // каталог основных ресурсов игры
│   ├── Animations // анимации
│   │   ├── Character
│   ├── Fonts // шрифты
│   ├── Resources // ресурсы (сохраненные составные части)
│   ├── Scenes // сцены
│   ├── Scripts // игровые скрипты
│   ├── Sprites // спрайты
│   ├── Textures // текстуры
├── Build // сборка игры
├── LICENSE
├── README.MD
└── images // скриншоты
```
